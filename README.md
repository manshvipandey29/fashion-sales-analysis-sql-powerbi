# Fashion Sales Analysis SQL Power BI

A complete fashion retail sales analysis project using SQL and Power BI. This project focuses on data cleaning, exploratory data analysis, SQL querying, and interactive dashboard creation to uncover sales trends, customer behavior, product performance, and business insights.

## Tools Used

* SQL
* Power BI
* Excel
